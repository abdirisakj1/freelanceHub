# 🚀 Freelance Marketplace

A modern freelance marketplace platform built with React and Supabase that connects clients with skilled professionals.

## ✨ Features

### For Clients

* Create and manage job postings
* Review applications from candidates
* View applicant profiles and proposals
* Direct messaging with applicants
* Real-time notifications
* Accept, reject, or complete applications
* Track application progress

### For Freelancers

* Browse available jobs
* Search and filter opportunities
* Submit applications with proposals
* Track application status
* Receive real-time notifications
* Direct messaging with clients
* Edit previously sent messages
* Manage professional profile

### Core Features

* Secure authentication
* Real-time updates
* Professional dashboard
* Responsive design
* Advanced search and filtering
* Notification center
* Relative timestamps (e.g., "2 min ago")
* Mobile-friendly experience

## 🛠️ Tech Stack
